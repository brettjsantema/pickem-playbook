# Underdog Guide — Project Brief

## What This Is
A publicly hosted referral/guide site covering DFS, sweepstakes casinos, and crypto prediction markets. Every site featured has a referral link attached. The tone is practical and helpful — real guides, not just ads.

## Dev Preferences
- No TypeScript
- No external UI libraries (no MUI, no Tailwind, no Bootstrap)
- Plain CSS — dark theme preferred
- Vite + React 18
- One-shot implementation style: build it complete, not in scaffolded stubs

## Site Structure

### Group 1 — Daily Fantasy / DFS Picks
- **Underdog Fantasy** (primary, full guide)
  - How to claim the deposit bonus
  - How to claim other ongoing offers
  - Step-by-step walkthrough with a referral signup CTA button
- **Novig** (secondary, recommendation section)
- **PrizePicks** (secondary, recommendation section)

### Group 2 — Sweepstakes Casinos
- Full explainer: what sweeps casinos are, how they work, how to navigate them
- Highlight 6–8 best sweeps casinos (each as a card)
  - Card includes: site name, brief description, referral link CTA button
- A couple paragraphs on general strategy/navigation tips

### Group 3 — Crypto Prediction Markets
- **Kalshi** (recommendation section)
- **Polymarket** (recommendation section)

## Layout Concept
- Single-page or tabbed layout with the three groups as major sections
- Each featured site gets a **card** component
- Underdog gets expanded card/section with full guide content
- Sweeps casinos: grid of cards, each with name, blurb, CTA button
- Referral links are the primary CTA on every card — styled as prominent buttons

## Key Goals
- Drive signups via referral links (these are monetized)
- Be genuinely useful — real bonus/offer guides, not filler
- Look clean and trustworthy enough to share publicly
- Mobile-friendly
